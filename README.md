# TrabalhoSOP
Primeiro trabalho de Sistemas Operacionais
Um progama escrito via códigos no bloco de notas e salvado .bat. Se consiste em apresentar opções para abrir ferramentas de busca, tendo como exemplo yahoo, google e bing. Apresenta o Menu informando os comandos mais importantes, e apresentando o comando --help caso queira ver todos os comandos, que é onde o comando --version se encontra.
